## 六、OpenClaw 核心能力

### 可用工具

- **文件操作**: read, write, edit, exec
- **浏览器控制**: browser (Chrome extension relay / headless)
- **消息发送**: message (多通道)
- **定时任务**: cron (定时提醒/自动化)
- **知识库**: feishu_wiki, feishu_doc, feishu_drive, feishu_bitable
- **网络**: web_search, web_fetch
- **语音**: tts, nodes (语音通话)
- **存储**: feishu_drive

### 配置的功能

- 消息通道: 飞书 (Feishu)
- 语言模型: MiniMax-M2.1
- 运行环境: Linux x64

## 七、常用命令

```bash
# OpenClaw 管理
openclaw gateway status      # 查看状态
openclaw gateway restart     # 重启
openclaw help                # 帮助

# 技能管理
clawhub search <keyword>     # 搜索技能
clawhub install <skill>      # 安装技能

# 环境
node --version               # Node 版本
npm --version                # npm 版本
python3 --version            # Python 版本
git --version                # Git 版本
```
