## 🔥 热门开源项目

### Claude Code (anthropics/claude-code)
> Claude Code 是一个代理编码工具，运行在终端中，能理解代码库，通过自然语言命令帮助你更快地编写代码、执行常规任务、解释复杂代码和处理 Git 工作流。

### Cloudflare Agents (cloudflare/agents)
> 在 Cloudflare 上构建和部署 AI Agents，支持无服务器架构。

### GitNexus (abhigyanpatwari/GitNexus)
> 零服务器代码智能引擎。完全在浏览器中运行，导入 GitHub 仓库或 ZIP 文件即可获得交互式知识图谱和内置 Graph RAG 代理。**今日获 424 stars**

### AI Tools System Prompts (x1xhlol/system-prompts-and-models-of-ai-tools)
> 收集了各种 AI 编码工具的系统提示，包括：Augment Code、Claude Code、Cursor、Devin AI、Trae、Windsurf 等。

### OpenBB (OpenBB-finance/OpenBB)
> 面向分析师、量化交易员和 AI 代理的金融数据平台。
